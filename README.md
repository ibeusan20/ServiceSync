# ServiceSync

ServiceSync is an Android app for browsing services and booking appointments, with user profiles, favorites, and a modern Jetpack Compose UI. Authentication and data storage are handled via Firebase.

## Features
- Sign up / Sign in (Firebase Auth)
- User profile management (edit profile details, map preferences, language, optional profile picture)
- Browse providers/services
- Appointment reservations (stored in Firestore)
- Favorites per user (stored in Firestore)
- Light/Dark theme (Material 3)

## Tech Stack
- Kotlin + Jetpack Compose (Material 3)
- Firebase: Authentication, Firestore
- (Optional) Profile image upload via Imgur API
